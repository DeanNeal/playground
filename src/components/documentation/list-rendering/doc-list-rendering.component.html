Conditional rendering

ac-for with arrays

	
    props: ()=>{
    	return {
    		items: [
	    		{name: 'Item1'}, 
	    		{name: 'Item2'}, 
	    		{name: 'Item3'}
    		]
    	}
    }
	
<ul><li ac-for="let (item, index) of item">{{index}}- {{item.name}}</li></ul>
will be replaces with 
  • {{index}}- {{item.name}}
ac-for with objects

	
    props: ()=>{
    	return {
	    	object: {
	    	    prop1: 'obj1',
	    	    prop2: 'obj2'
	    	}
    	}
    }
	
<ul><li ac-for="let (item, key, index) of item">{{index}}- {{key}} {{item}}</li></ul>
  • {{index}}- {{key}} {{item}}
